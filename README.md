## О проекте

## Начиная

### Предпосылки

- npm
  ```sh
  npm install npm@latest -g
  ```

### Установить

1. Получите бесплатный ключ API на [NewsAPI](https://newscatcherapi.com/)
2. Клонируйте репозиторий

   ```sh
   git клон https://github.com/your_username_/Project-Name.git
   ```

3. Установите пакеты NPM

   ```sh
   npm install
   ```
